# 😊 Sentiment Analysis Dashboard

A modern AI-powered web application that analyzes the sentiment of user text using Python, Flask, and TextBlob. The application provides real-time sentiment analysis with an interactive dashboard and data visualization.

## 🚀 Features

- Real-time sentiment analysis
- Positive, Neutral, and Negative prediction
- Interactive progress bar
- Emoji-based sentiment display
- Statistics dashboard
- Live Pie Chart (Chart.js)
- Analysis history
- Modern responsive UI

## 🛠️ Technologies Used

- Python
- Flask
- TextBlob
- HTML5
- CSS3
- JavaScript
- Chart.js

## ▶️ Installation

```bash
git clone <repository-url>
cd Sentiment-Analysis-Dashboard

python -m venv venv
venv\Scripts\activate

pip install -r requirements.txt

python -m textblob.download_corpora

python app.py
```

Open:

http://127.0.0.1:5000
